## L01_Threshold（门槛）

**主题：** 学会质疑，一切并非表象所示。

**流程：**
1. 玩家在起始区域内，场景中有若干可交互物品
2. 交互物品后触发弹窗文本提示（引导玩家理解"反转"概念）
3. 完成特定交互后，移除通往 L02 必经之路上的障碍物
4. 玩家通过障碍物缺口进入 L02 触发区

**出口触发：** 障碍物消除后，通道末端放置关卡切换触发器 → 加载 `L02_Deception`

**关键交互物：**
- 可交互物品（数量待定，建议 2–3 个）：触发弹窗提示
- 障碍物：初始阻塞通道，条件满足后消失

**设计备注：**
- 此关为教学关，弹窗文本应点明游戏核心主题："Trust Nothing"
- 不需要战斗，节奏应平缓

## L02_Deception（欺骗）

**主题：** 眼见不为实——怪物是宝箱，宝箱是怪物。

**流程：**
1. 玩家交互入口大门 → 大门本身不动，但门所在的**墙体**旋转打开
2. 进入后，场景中同时出现"怪物"模型和"宝箱"模型
3. 实际上：
   - **怪物模型** = 宝箱（不攻击，被击败后可打开，内含收藏品）
   - **宝箱模型** = 怪物（会主动攻击玩家）
4. 玩家通过攻击击杀"宝箱怪物"，打开"怪物宝箱"获取收藏品
5. 获取收藏品后移除通往 L03 的障碍物

**出口触发：** 收藏品获取后，障碍物消除 → 通道末端关卡切换触发器 → 加载 `L03_WalkOnAir`

**关键交互物：**
- 入口大门（交互物）：触发墙体旋转动画
- 宝箱怪物（战斗单位）：具有攻击行为，被击杀后障碍物消除
- 怪物宝箱（可交互单位）：无攻击，被"击败"后触发开箱动画，给予收藏品

**设计备注：**
- 墙体旋转而非门本身开启，是第一次视觉层面的反转提示
- 收藏品的获取流程要清晰：先打宝箱（怪），再开怪物（箱）

## L03_WalkOnAir（凌空）

**主题：** 脚下是虚空，空气才是路。

**流程：**
1. 玩家交互入口大门 → 大门开始 **360° 持续旋转**（不停止）
2. 进入后，场景呈现"浮空平台"和大片"空气"区域
3. 实际碰撞：
   - **浮空平台** = NoCollision（玩家直接穿过/坠落）
   - **空气区域** = BlockAllDynamic（玩家可以在"空气"上行走和跳跃）
4. 玩家在"空气"上跳跃前进，越过关卡到达对岸
5. 到达对岸触发 L04 开启

**出口触发：** 到达对岸触发区 → 加载 `L04_Kindled`

**关键Actor：**
- 入口大门：交互后触发持续旋转动画（不锁止）
- `BP_FakeFloor`（浮空平台）：Visible，NoCollision
- `BP_HiddenPlatform`（空气路径）：Hidden In Game，BlockAllDynamic

**设计备注：**
- 大门 360° 旋转是对"进入就没有回头路"的隐喻，也是视觉上的反转信号
- 空气路径的布局应有一定引导性（如光效或粒子），让玩家能够发现规律
- 坠落应有复活点或重置机制

## L04_Kindled（化身）

**主题：** 成为工具本身。

**流程：**
1. 玩家交互入口大门 → 大门**直接倒下**（向前方倒塌作为地板）
2. 进入后，场景中分布三把火炬和一把中央宝剑
3. 交互**第一把火炬**：玩家外观立刻变为火炬模型，但保留全部移动/跳跃能力
4. 每交互一把后续火炬：玩家（火炬体）上额外附加一根火炬模型（视觉叠加）
5. 集齐三把火炬后，中央宝剑解锁（变为可交互状态）
6. 与宝剑交互：玩家外观变为宝剑模型
7. 宝剑交互完成后，L05 解锁

**出口触发：** 宝剑交互完成 → 解除 L05 封锁 → 触发区加载 `L05_Unsheathed`

**关键Actor：**
- 入口大门：交互后播放倒塌动画
- 火炬 × 3：顺序或自由交互均可，每次交互更新玩家外观
- 中央宝剑：初始锁定，三把火炬集齐后解锁

**设计备注：**
- 玩家变成火炬后的外观叠加是本关视觉笑点，确保动画过渡流畅
- 宝剑外观需延续至 L05 全程，注意跨关卡保存玩家状态
- 大门"倒下作为地板"是本关的第一个视觉反转

## L05_Unsheathed（出鞘）

**主题：** 万物皆为怪物，唯剑不欺。

**玩家状态：** 全程保持 L04 结束时的**宝剑外观**

**流程：**
1. 玩家交互入口大门 → 大门出现**血条**，变为需击杀的怪物
2. 击杀大门怪物后，前方**巨大钥匙**出现交互提示"开始挑战"
3. 交互巨大钥匙 → 钥匙变为怪物，玩家击杀
4. 击杀后，**结算方碑**出现交互提示
5. 交互结算方碑 → 方碑也变为怪物，玩家击杀
6. 击败方碑怪物后，直接进入 L06，玩家外观恢复正常

**出口触发：** 方碑怪物被击败 → 自动加载 `L06_Revelation`，同时重置玩家外观

**关键Actor：**
- 入口大门（怪物化）：具备血条和战斗AI
- 巨大钥匙（怪物化）：交互触发变身，触发战斗
- 结算方碑（怪物化）：交互触发变身，最终战斗

**设计备注：**
- 本关所有"正常物件"最终都会变成怪物，强化"Trust Nothing"主题
- 三段战斗节奏：门→钥匙→方碑，强度可依次递增
- 进入 L06 时恢复玩家默认外观（清除 L04 设置的外观状态）

## L06_Revelation（启示）

**主题：** 真相在终点等待。

**流程：**
1. 玩家沿一条狭窄小路前行（无战斗，无障碍）
2. 小路尽头有一个 NPC
3. 玩家与 NPC 交互，触发对话序列
4. 对话完成后 → **游戏胜利**，显示结局画面/结算UI

**出口触发：** 对话完成 → 触发胜利事件

**关键Actor：**
- NPC（带对话组件）
- 胜利触发器（对话完成后激活）

**设计备注：**
- 小路应有一定长度，给玩家"走向终点"的仪式感
- NPC 的对话内容由叙事团队提供（Jiawei 负责文案）
- 结局画面/UI 由 UI 组负责（Phase 4）
